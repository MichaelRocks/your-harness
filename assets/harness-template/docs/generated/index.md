# Generated Artifacts

Use this area for generated outputs that support the repository but should not
inflate the hand-written docs.

## Examples

- Generated API snapshots
- Schema exports
- Tool-produced architecture diagrams
- Machine-generated reports used for review or validation

## Rules

- Treat generated content as derivative, not as the main source of truth.
- Link back to the durable hand-written doc that explains why the artifact
  matters.
- Note regeneration steps elsewhere if the process is not obvious.
