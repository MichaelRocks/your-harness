# Active Plans

Create new in-progress change plans here.

- Medium changes use a single file named `YYYY-MM-DD-short-slug.md` with a
  concise planned-change summary plus an explicit task checklist.
- Large changes use a folder named `YYYY-MM-DD-short-slug/` containing
  `overview.md` plus one file per phase. The overview and every phase file
  should include explicit tasks, and tasks should be marked done as work
  completes.
